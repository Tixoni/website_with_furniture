Проект представляет собой статическую веб-страницу, сверстанную по предоставленному макету.  
Работа выполнялась поэтапно в рамках практических занятий:

- **Практическое занятие 2** — создание семантической HTML-структуры страницы

## Семантическая структура страницы

Страница построена с использованием семантических HTML-тегов:

1. **`<header>`** — шапка сайта с логотипом и навигационным меню
2. **`<section>`** — основные смысловые блоки страницы:
   - Hero-секция (приветственный экран)
   - Блок преимуществ бренда
   - Секция товаров
   - Популярные товары
   - Подписка на рассылку
   - История бренда
3. **`<footer>`** — подвал сайта с навигацией, формой подписки и социальными ссылками

## Реализация Практического занятия 3

В рамках практического занятия №3 были выполнены следующие задачи:

### Подключены стилии текста

### Навигация по якорным ссылкам
- Реализована навигация по разделам страницы с использованием якорей (`id`)
- Пункты меню соотнесены с существующими секциями страницы
- Добавлена плавная прокрутка с помощью CSS

```css
html {
  scroll-behavior: smooth;
}
````

### Интерактивные состояния элементов

Добавлены CSS-состояния для интерактивных элементов интерфейса:

* `:hover` — для ссылок, кнопок и карточек товаров
* `:focus` — для ссылок, кнопок и полей ввода
* `:active` — для кнопок

Это улучшает пользовательский опыт и доступность интерфейса.

### Микро-анимации

* Реализована анимация плавного появления секций при загрузке страницы
* Анимации выполнены средствами CSS без использования JavaScript
