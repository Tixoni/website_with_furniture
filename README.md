
# Верстка сайта по макету на HTML

**Ссылка на github pages :[https://tixoni.github.io/website_with_furniture/]**
**Практическое занятие:** 2  
**Тема:** Верстка сайта по макету(вариант 2) на HTML

---

## Описание проекта

Проект представляет собой статическую веб-страницу, сверстанную по предоставленному макету с использованием семантической HTML-разметки. Цель работы — освоить принципы создания корректной структуры веб-страницы, работу с типографикой, валидацию кода и публикацию проекта на GitHub Pages.

---

## Используемые технологии

- **HTML5** — семантическая разметка
- **CSS** — базовые стили (в рамках следующего занятия)
- **Git** — контроль версий
- **GitHub Pages** — хостинг проекта

---

## Структура проекта

```
project/
├── index.html
├── README.md
├── assets/
│   ├── img/
│   │   └── (изображения из макета)
│   ├── icons/
│   │   └── favicon.ico
│   └── fonts/ (будет добавлено позже)
├── css/
│   └── style.css (будет добавлено в Практическом занятии 3)
└── js/ (опционально)
```

---

## Семантические блоки страницы

1. **`<header>`** — шапка сайта с логотипом и навигацией
2. **`<main>`** — основное содержимое, разделенное на секции:
   - Секция 1: Герой / Приветствие
   - Секция 2: О нас
   - Секция 3: Услуги / Товары
   - Секция 4: Портфолио
   - Секция 5: Контакты
3. **`<footer>`** — подвал сайта с контактной информацией и ссылками

---

## Проверки и валидация

- HTML-код проверен через [Validator W3C](https://validator.w3.org/)
- Текст обработан через онлайн-типограф для корректного отображения кавычек и тире
- Изображения оптимизированы с помощью [Squoosh.app](https://squoosh.app/)
